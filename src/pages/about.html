Understanding the machine learning technology behind our heart disease prediction system

01. Problem Definition

"Given clinical parameters about a patient, can we predict whether or not they have heart disease?"

Cardiovascular disease is the leading cause of death globally, accounting for 31% of all deaths worldwide. Early detection and intervention can significantly improve patient outcomes and save lives.

Our machine learning model aims to assist healthcare professionals by providing accurate predictions based on readily available clinical parameters, enabling earlier intervention and better patient care.

05. How the Model Works

1

Data Input & Preprocessing

Patient clinical data (13 features) is collected through our user-friendly interface. The data is then preprocessed and normalized to match the format used during model training.

Input → Validation → Normalization → Feature Vector [13 dimensions]
2

Logistic Function Application

The model applies the logistic (sigmoid) function to compute a probability score between 0 and 1, representing the likelihood of heart disease presence.

P(heart disease) = 1 / (1 + e^-(β₀ + β₁x₁ + β₂x₂ + ... + β₁₃x₁₃))
3

Classification & Confidence Score

Based on the probability score and optimized decision threshold, the model classifies the patient as either "Heart Disease Present" or "No Heart Disease" and provides a confidence percentage.

If P < 0.5
No Heart Disease Detected
If P ≥ 0.5
Heart Disease Present
4

Results Interpretation

The system generates a comprehensive report showing the prediction, confidence level, and key contributing factors from the patient's data that influenced the decision.
